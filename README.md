#  Real-Time Satellite Tracker

A web-based satellite tracking application that visualizes satellites orbiting Earth in real time using **JavaScript** and **CesiumJS**.

## Features

*  Interactive 3D Earth visualization
*  Real-time satellite tracking
*  Search satellites (ISS, Starlink, etc.)
*  Displays satellite information and orbit position
*  Fast and lightweight web application

##  Technologies Used

* HTML
* CSS
* JavaScript
* CesiumJS
* satellite.js
